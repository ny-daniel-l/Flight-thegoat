# Styleguide – Flight vs Cash
Das Logo besteht aus dem Schriftzug:

**FLIGHT VS CASH**

Flight wird orange dargestellt, Cash blau. Dadurch erkennt man direkt die zwei Seiten der Rivalität.

## Farben

| Zweck | Farbe | Hex |
|---|---|---|
| Hintergrund | Schwarz | `#0a0a0a` |
| Karten | Dunkelgrau | `#141414` |
| Flight-Akzent | Orange | `#ff6b00` |
| Cash-Akzent | Blau | `#00c2ff` |
| Highlight | Gold | `#ffd700` |
| Text | Hellgrau | `#f2f2f2` |
| Sekundärtext | Grau | `#888888` |

## Typografie

- Titel: `Bebas Neue`
- Fliesstext: `Barlow`
- Navigation: Grossbuchstaben, fett, mit Abstand zwischen Buchstaben

## Abstände

- Seitenabstand Desktop: `32px`
- Seitenabstand Mobile: `18px`
- Section Padding Desktop: `90px 32px`
- Section Padding Mobile: `70px 18px`
- Karten Padding: `24px`

## Layout

### Desktop

- Header oben fixiert
- Navigation rechts
- Inhaltsbereich maximal `1100px`
- Profilbereich mit zwei Spalten
- Rasterseite mit drei Spalten
- Formulare: Label links, Feld rechts

### Mobile Breakpoint

Breakpoint: `768px`

Auf kleinen Bildschirmen:

- Navigation wird unter dem Logo angezeigt
- Raster werden einspaltig
- Formulare werden untereinander dargestellt
- Schriftgrössen werden kleiner
- Bilder und Videos sind 100% breit
