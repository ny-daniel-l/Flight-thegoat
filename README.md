# Flight vs Cash Website

Diese Webseite handelt von der bekannten Basketball-YouTube-Rivalität zwischen FlightReacts und CashNasty. Besucher erhalten eine kurze Einführung, Informationen zu beiden Creators, eine Rasterseite mit wichtigen Rivalry-Momenten und mehrere Formulare wie Registrierung, Login und Passwort vergessen. Die Webseite verwendet Flexbox für die Hauptstruktur und CSS Grid für die Rasterseiten. Zusätzlich ist die Seite responsive aufgebaut und enthält semantische Medienbereiche mit Bild, Figure, Figcaption und Video.

## Seiten

- `index.html` – Projekt-Intro mit Bild, Text und Video
- `about.html` – About-Seite mit Listen
- `creators.html` – Rasterseite mit Creator- und Rivalry-Karten
- `register.html` – Registrierungsformular
- `login.html` – Pseudo-Login
- `password-forgot.html` – Passwort zurücksetzen
- `confirm.html` – Bestätigungsseite

## Layout-Techniken

- Hauptlayout: Flexbox
- Rasterseiten: CSS Grid
- Responsive Design: Media Query bei `max-width: 768px`

## Publikation

Die fertige Webseite kann z.B. mit GitHub Pages, GitLab Pages, Netlify oder FTP veröffentlicht werden.

## Optimierung

Mit `npm run build` werden Zielordner gelöscht, SCSS zu CSS kompiliert, HTML/CSS minimiert und Assets kopiert.
